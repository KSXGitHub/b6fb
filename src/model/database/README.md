# Notes

Create `database.php` in this folder with the following structure:

```php
<?php
return [
  'domain' => '<Domain>',
  'username' => '<Database Admin Username>',
  'password' => '<Password>',
  'dbname' => '<Database Name>',
];
?>
```

**Example:**

```php
<?php
return [
  'domain' => 'localhost',
  'username' => 'root',
  'password' => '',
  'dbname' => 'b6fb',
];
?>
```
